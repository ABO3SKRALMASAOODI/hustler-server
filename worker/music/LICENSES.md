# Built-in music library — licences

Every track below is CC0 1.0 (public domain dedication): no attribution
is required and redistribution inside a customer's exported video is
permitted. This file is the paper trail.

Each source URL was fetched and checked to declare
`creativecommons.org/publicdomain/zero/1.0` with no restrictive
`licenses/*` URL anywhere on the page — 24/24 verified.

Moods are our own editorial labels; the source does not state them.
Audio was loudness-normalised to about -16 LUFS and trimmed of
leading/trailing silence, so `gain_db` means the same on every track.
